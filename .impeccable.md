# Impeccable Design Context

## Design Context

### Users
面向大众用户发布的 Chrome 新标签页扩展。用户每天打开新标签页几十次，核心场景是：
快速向多个 AI 平台发问、点开常用链接。设置是低频行为，但首次配置体验决定留存。
不能假设用户懂技术（不知道什么是 URL、JSON、favicon）。

### Brand Personality
有个性、有编辑感（editorial）、克制。三个词：精致、笃定、安静。
每天见面的页面，应当像一本好杂志的扉页：有排版气质，不喧哗。

### Aesthetic Direction
- 在现有黑白极简基础上注入编辑感：更有记忆点的字体排印、非对称构图、克制的留白节奏。
- 反参考：通用 AI 风格（玻璃拟态滥用、千篇一律的胶囊按钮、渐变文字）。
- 深浅双主题，跟随系统 + 手动覆盖。
- 背景视频/图片之上的内容必须始终可读。

### Design Principles
1. 高频操作就地完成（链接增删、背景切换在新标签页内），低频配置进设置页。
2. 大众用户优先：空状态要教学、错误要友好、危险操作可撤销，不暴露技术术语。
3. 排版即设计：用字体层级和间距节奏建立个性，而不是堆装饰。
4. 一次只让用户做一个决定：渐进披露，默认值即最佳实践。
5. 所有反馈走统一的轻量通知，不用 alert()。
